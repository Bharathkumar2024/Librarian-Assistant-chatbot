# The Human Made - Liberin Game Chat

This project is a game chat application built with React, TypeScript, and Vite.

## Project info

This project is designed to be a high-quality, human-made application.

## Getting Started

To get started with this project locally:

1.  Clone the repository.
2.  Install dependencies:
    ```sh
    npm install
    ```
3.  Start the development server:
    ```sh
    npm run dev
    ```

## Technologies Used

-   Vite
-   TypeScript
-   React
-   shadcn-ui
-   Tailwind CSS
